# Reproducibility Beyond the Code – organizing research projects in R

## What this workshop covers

Reproducibility is now a baseline expectation in scientific research, yet many projects fall short because they lack clear structure, documentation, and organization. This workshop covers three broad themes:

1.  **Organizing R projects & repositories** — logical directory structure, separating raw data / processed data / scripts / outputs, and writing scripts that are readable and rerunnable.
2.  **Documentation** — effective READMEs and lightweight data dictionaries.
3.  **Git & GitHub** — version control basics, recovering earlier versions, safely refactoring or deleting code, `.gitignore`, and GitHub as a tool for meeting journal data/code-availability requirements.

No prior Git or R experience is required, but basic familiarity with research and data analysis is assumed.

## Repository structure

```         
Data/       # raw, unmodified source data (gitignored)
Derived/    # outputs generated by scripts (recreatable, gitignored)
Figures/    # saved plots (gitignored unless a static/final asset)
Scripts/    # analysis code
  qmd/      # Quarto slides
```

## Usage

`01_registrant_map.R` and `qmd/02_slides.qmd` are fully reproducible from what's in this repo. `00_wrangling.R` is listed for context but isn't included — it reads the raw registrant workbook, which is gitignored because it contains real names, emails, and addresses.

- `00_wrangling.R` (local only, not in this repo) - Creates the anonymized, shareable location dataset (first name + last initial, city/state/country) from the raw registrant workbook
- `01_registrant_map.R` — Reads that dataset and plots a US state choropleth of where registrants are from
- `qmd/02_slides.qmd` - Slides written in Quarto

## Column names and descriptions

`first_name:` Participants' first name

`last_initial:` First initial of participants' last name

`city:` City where participant resides

`state:` State where participant resides

`country:` Country where participant resides
